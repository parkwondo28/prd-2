## 기술적 고려사항

### 현재 구현 상태

1. **지도 API**: 네이버 지도 스크립트 로드 구조 완성, 실제 API 키는 환경 변수로 설정 필요
2. **데이터 저장**: 현재는 메모리 기반 임시 데이터 사용, 추후 데이터베이스 연동 필요
3. **필터링**: 클라이언트 측 필터링 로직 구현 완료

### 향후 개선 사항

1. **데이터베이스 연동**: PostgreSQL 또는 MongoDB 연동
2. **인증 시스템**: 사용자 인증 및 권한 관리
3. **이미지 업로드**: 카페 사진 업로드 기능
4. **검색 기능**: 카페 이름/주소 검색 기능
5. **페이지네이션**: 카페 목록 페이지네이션

## 환경 변수 설정 필요

프로젝트 루트에 `.env.local` 파일을 생성하고 다음 변수를 설정해야 합니다:

```
NEXT_PUBLIC_NAVER_MAP_CLIENT_ID=your_naver_map_client_id
```

네이버 클라우드 플랫폼에서 지도 API 클라이언트 ID를 발급받아 설정하세요.

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.
